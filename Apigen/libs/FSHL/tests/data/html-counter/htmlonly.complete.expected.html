 49:     <section id="overview">
 50:         <div>
 51:             <p><a href="http://github.com/kukulich/fshl/zipball/master" onclick="_gaq.push(['_trackEvent', 'Click', 'Download', 'Latest']);" class="button">Download FSHL</a></p>
 52: 
 53:             <p><a href="https://github.com/kukulich/fshl" onclick="_gaq.push(['_trackEvent', 'Click', 'Repository', 'Ribbon']);" id="fork"><span>Fork FSHL</span></a></p>
 54: 
 55:             <p>FSHL is a free, open source, universal, fast syntax highlighter written in PHP. A very fast parser performs syntax highlighting for few languages and produces a HTML output.</p>
 56: 
 57:             <p>FSHL library is a simple, easy to use syntax highlighter. Its API provides only one method that is really need to highlight sources and three auxiliary methods to sets lexer and output mode.</p>
 58: 
 59:             <p>FSHL core is very flexible and it is very easy to add new languages. Feel free to do so and do not forget to share them with the rest of the world.</p>
 60:         </div>
 61:     </section>

 63:     <section id="installation">
 64:         <h2>Installation</h2>
 65:         <div>
 66:             <p>FSHL should be installed using the <a href="http://pear.php.net/">PEAR Installer</a>. This installer is the backbone of PEAR, which provides a distribution system for PHP packages, and is shipped with every release of PHP since version 4.3.0.</p>
 67: 
 68:             <p><a href="http://pear.kukulich.cz" onclick="_gaq.push(['_trackEvent', 'Click', 'PEAR channel']);">The PEAR channel</a> that is used to distribute FSHL needs to be registered with the local PEAR environment.</p>
 69:             <pre>pear channel-discover pear.kukulich.cz</pre>
 70: 
 71:             <p>This has to be done only once. Now the PEAR Installer can be used to install packages from the kukulich channel:</p>
 72:             <pre>pear install kukulich/FSHL</pre>
 73: 
 74:             <p>After the installation you can find the FSHL source files inside your local PEAR directory.</p>
 75:         </div>
 76:     </section>

 78:     <section id="example">
 79:         <h2>Example</h2>
 80:         <div>
 81:             <pre><code><span class="xlang">&lt;?php</span>
 82: <span class="php-var">$highlighter</span> = <span class="php-keyword1">new</span> \FSHL\Highlighter(<span class="php-keyword1">new</span> \FSHL\Output\Html());
 83: <span class="php-var">$highlighter</span>-&gt;setLexer(<span class="php-keyword1">new</span> \FSHL\Lexer\Php());
 84: <span class="php-keyword1">echo</span> <span class="php-quote">'&lt;pre&gt;'</span>;
 85: <span class="php-keyword1">echo</span> <span class="php-var">$highlighter</span>-&gt;highlight(<span class="php-quote">'&lt;?php echo &quot;Hello world!&quot;; ?&gt;'</span>);
 86: <span class="php-keyword1">echo</span> <span class="php-quote">'&lt;/pre&gt;'</span>;
 87: <span class="xlang">?&gt;</span></code></pre>
 88:             <p>Or</p>
 89: 
 90:             <pre><code><span class="xlang">&lt;?php</span>
 91: <span class="php-var">$highlighter</span> = <span class="php-keyword1">new</span> \FSHL\Highlighter(<span class="php-keyword1">new</span> \FSHL\Output\Html(), \FSHL\Highlighter::OPTION_TAB_INDENT | \FSHL\Highlighter::OPTION_LINE_COUNTER);
 92: <span class="php-keyword1">echo</span> <span class="php-quote">'&lt;pre&gt;'</span>;
 93: <span class="php-keyword1">echo</span> <span class="php-var">$highlighter</span>-&gt;highlight(<span class="php-quote">'&lt;?php echo &quot;Hello world!&quot;; ?&gt;'</span>, <span class="php-keyword1">new</span> \FSHL\Lexer\Php());
 94: <span class="php-keyword1">echo</span> <span class="php-quote">'&lt;/pre&gt;'</span>;
 95: <span class="xlang">?&gt;</span></code></pre>
